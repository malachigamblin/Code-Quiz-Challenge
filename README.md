# 
Web API Coding Quiz

## 
This code is used for taking a coding quiz that uses JavaScript for funcionality. When the start button is pressed, a timer will begin to countdown from 60 seconds and a question will be presented. After a question is answered, another question will be presented. If a question is answered incorrectly, the timer will subtract an extra 5 seconds. When all the questions have been answered or the timer reaches 0, the quiz will be over. The score from the quiz will be presented, with the option to enter in initals.
